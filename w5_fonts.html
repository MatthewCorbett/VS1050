Task 2 - Visual Formatting with CSS (embedded)

Open the page you created in Lab 2 (My Favourite Things - index.html). If you can't find it, or didn't save it, you can use the page on Canvas (download the file from Week 5 - favourite.html) Now that your Web page has structure, you can add visual formatting suggestions using CSS.

  1. In the content of the <head> tag, place a CSS <style> tag: 
    <style type="text/css">
</style>
  2. Inside this <style> tag, add selectors for body and address tags: 
      body { }
  address { }
  3. Inside the braces ({ }) add some property-value pairs. For example, to make most text purple on a light blue background and aligned to the left of the page use body { color: purple; background-color: lightblue; text-align: left }
  4. To make text inside the address block forest green and aligned to the right of the page use address { color: forestgreen; text-align: right }
  5. Reload the Web page into your Web browser to see the changes in visual formatting.
  6. Add more style rules to change the formatting of level-1 and level-2 headings.
  7. Add Pseudo-class selectors for the email you have in <address> block. You should define pseudo classes so that your email address appears differently when it is unvisited, visited, hovered and selected .

Task 3 - More CSS Styles (Using external style sheet)

External sheets are ideal for maintaining formatting consistency over many pages on a site. This task wants you to experiment with as many different styles as you can. See if you can implement the following styles, and watch their effect in the browser...

  1. Create an external style sheet and link it in the HTML file using the link tag. The following tasks should be completed using the external style sheet.
  2. Change the margin of the webpage to 0px. Change it to 50px.
  3. Change the type of bullets used for lists. Better yet, change the bullet to an image.
  4. Add styles to insert a border around the image. Modify margin, padding, border-width, border-style and border-color properties.
  5. Make all level-1 headings centre-aligned
  6. Insert a background image and repeat it throughout the page.
  7. Set a background color (highlighting) behind all paragraphs. Try to set colour using both rgb and hexadecimal schemes.
  8. Now use the same external style sheet to format both favourite pages you created in Lab 2 (favourite01.html and favourite02.html). Change some properties in your external style sheet and see how it is reflected in both favourite pages.

Task 4 - Formatting with CSS Selectors

Open the copy page you created in Lab 2 (My Favourite Things - index2.html). Now you will add visual formatting to this page using CSS selectors and watch their effect in the browser...

  1. In the content of the <head> tag, place a <style> tag as you have done in Task 2.
  2. Inside the <style> tag, add the following selectors
    • One ID selector with background-color property. And use the id attribute in the target paragraph to receive the formatting.
    • One class selector with color property. And use that class name to format different tags in your index2.html file. For example one paragraph and one list.
    • Try to test one of the descendant and/or sibling selectors (Optional).

Task 5 - Cascading multiple style declaration

In this task you see how the styling rules defined at different levels are cascaded in CSS. Open the file you created in Task 3 using external style sheet.

  1. Insert a few document level styles (inside the head element) to override the same styles defined in external style sheet with different values. For example define a different style for the bullets to override the style defined in the external style sheet.
  2. Check how the document level styles override the external styles. If you swap the order of the style and link tags in the head element, notice how the cascade changes. Why does this occur?
  3. Now override the document level styles you added in Step 1 using in-line style declaration.
  4. Check how in-line styles override the document level styles.
